# World Cup 2026 Ticker

Static site for the World Cup 2026 ticker.

## Local development

- `npm start`
- Open `http://localhost:4173`

## Deployment

This repository is configured to deploy to GitHub Pages through the workflow in `.github/workflows/deploy-pages.yml`.

- Push to `main` to trigger a deployment.
- In the GitHub repository settings, set Pages to use `GitHub Actions` as the source if it is not already enabled.
- The published site URL should be `https://daveweij.github.io/wc2026-ticker/`.
